xpipeman

Historical source code version 1.5 patched for compiling on "latest" ubuntu linux (20.04 / 22.04 / 24.04 / ...)

Xpipeman is a classic pipe-connecting puzzle game

“Xpipeman is a game of skill requiring the user to connect together pieces of a pipe to allow a liquid to flow through without leaking out”


Get the source code from:

GitLab GitHub

Or, play it directly in your browser! Try the following (WIP) AI port to SDL2 converted then to WebAssembly! with Emscripten

Controls

  • Move a pipe: W A S D or H J K L or
  • Place a pipe: P or Space or or double
  • Fast flow: Tab or Fast Flow bottom button
  • Start a new game: N or New Game bottom button
  • Quit (actually go to initial screen): Ctrl + C or Quit bottom button
  • Close popups: Esc or Space or Enter or popup bottom button
  • Quit, New Game, Score, Info and Fast Flow bottom labels are clickable buttons
