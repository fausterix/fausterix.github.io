xpipeman

Historical source code version 1.5 patched for compiling on "latest" ubuntu linux (20.04 / 22.04 / 24.04 / ...)

Xpipeman is a classic pipe-connecting puzzle game

“Xpipeman is a game of skill requiring the user to connect together pieces of a pipe to allow a liquid to flow through without leaking out”


Get the source code from:

GitLab GitHub

Or try the following port to (SDL2 converted to) webasm (WIP, and maybe not mobile friendly 🤷)

Controls

  • Move a pipe: X or Right Arrow, Z or Left Arrow, "e; or Up Arrow, / or 7 or Down Arrow
  • Move a pipe: Left Click
  • Place a pipe: P, Space, Middle Click, or Double Left Click
  • Fast flow: Tab
  • Start a new game: N
  • Quit: Ctrl+C —actually do nothing here
  • Quit, New Game, Score and Info labels on the bottom are clickable
